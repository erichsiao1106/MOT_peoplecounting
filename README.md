### Before running
#### prepare following items
* 1. mp4 file
* 2. yolo's weights
* 3. yolo's cfg file
* 4. find location of above files in `main.py` and specify with their right path
---
### Run `counting_main.py`
---
### About input format
* The inputs for tracker is a list of bounding boxes
* Boxes are defined as [x, y, w, h]
